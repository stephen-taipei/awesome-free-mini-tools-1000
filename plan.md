# 1000 個免費前端工具開發計畫

> 本文件為總覽與索引，詳細工具清單請參閱各分部計畫檔案

## 計畫概述

- **目標**：開發 1000 個純前端小工具
- **分類**：25 大類，每類 40 個工具
- **開發方式**：每日批量開發 5-10 個工具
- **預估週期**：100-200 個工作日

## 技術標籤說明

每個工具都會標註使用的技術，方便查詢：

| 標籤 | 說明 |
|-----|------|
| `WASM` | 使用 WebAssembly 加速 |
| `WebGPU` | 使用 GPU 運算 |
| `Canvas` | 使用 Canvas 2D API |
| `WebGL` | 使用 WebGL 3D 渲染 |
| `Worker` | 使用 Web Worker 背景運算 |
| `Audio` | 使用 Web Audio API |
| `Video` | 使用影片處理 |
| `File` | 使用 File API |
| `Crypto` | 使用加密 API |
| `Storage` | 使用本地儲存 |
| `PWA` | 支援離線使用 |
| `i18n` | 支援多國語言 |

## 難度等級說明

| 等級 | 說明 | 預估開發時間 |
|-----|------|-------------|
| L1 | 簡單：基礎 API 使用 | 0.5-1 小時 |
| L2 | 中等：需整合多個 API | 1-2 小時 |
| L3 | 進階：需使用 WASM/WebGPU | 2-4 小時 |
| L4 | 複雜：需自訂演算法 | 4-8 小時 |
| L5 | 高級：需深度優化 | 8+ 小時 |
